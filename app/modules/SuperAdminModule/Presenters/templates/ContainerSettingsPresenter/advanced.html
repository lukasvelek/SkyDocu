Container settings

$component sidebar$

Create technical account

This will create a technical account for given user. The user is automatically added to the container Administrators group.

$CONTAINER_NEW_TECHNICAL_ACCOUNT_FORM_LINK$



Run migrations

This will run migrations on this container. The container will be turned off and turned back on after the migrations finish running.

$CONTAINER_RUN_MIGRATIONS_LINK$



Remove container from distribution

This will permanently remove the container from distribution. That means that the container database schema will not be updated automatically during SkyDocu new version update.

$CONTAINER_REMOVE_FROM_DISTRIBUTION_LINK$



Delete container

This will permanently delete the container and all its data. Created users will remain created and have to be remove manually.

$CONTAINER_DELETE_LINK$
